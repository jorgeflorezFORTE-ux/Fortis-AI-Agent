# Fortis 3.0 — Asesor Financiero + Contable + Libertad Financiera

## 3 Módulos

### 💰 Asesor Financiero
- Dashboard con ingresos vs gastos por empresa y personal
- Centro de carga: sube CSVs y se clasifican solos por 4 dígitos de cuenta
- Drill-down por categoría con cada transacción
- Asesor AI: analiza gastos y dice exactamente qué recortar
- Chat financiero en lenguaje natural

### 📋 Asesor Contable
- Reconciliación banco vs QuickBooks
- Detecta gastos mal clasificados entre empresas
- Detecta gastos personales mezclados con negocio
- AI sugiere journal entries exactos para QB
- Optimización de taxes (QBI 20%, S Corp, SALT)

### 🎯 Libertad Financiera
- FIN Score: qué % del meta $20K/mes tienes
- Tracker de activos (propiedades, CDs, negocios)
- Tracker de deudas (hipotecas, tarjetas, préstamos)
- Net worth en tiempo real
- Scorecard mensual A-F por área

## Instalación

```bash
npm install
cp .env.example .env.local  # ← pega tu ANTHROPIC_API_KEY
npm run dev
```

Abre http://localhost:3000
